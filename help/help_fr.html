Conseils rapides

  • Appuyez sur cette icône pour accéder au menu Global qui vous permet :
    • d'effectuer une recherche avec SpeedSearch (Saleslogix 8.0 et versions ultérieures) ;
    • d'ajouter rapidement un compte ou un contact grâce aux actions rapides ;
    • d'accéder à tous les types de fiches du client Mobile grâce à la fonction Accéder ;
    • de modifier le contenu du menu Accéder et définir l'ordre d'affichage de ses éléments avec la fonction Configurer le menu. Le premier élément de la liste de l'écran Configurer le menu devient votre vue par défaut et est chargé en premier une fois que vous êtes connecté ;
    • d'accéder aux paramètres pour effacer les identifiants enregistrés ou le stockage, ainsi que pour accéder aux journaux d'erreurs.
      • L'option Effacer le stockage de la section Paramètres est un simple outil de dépannage permettant d'effacer la configuration stockée en local.
    • accéder à l'aide ;
    • vous déconnecter.
  • Appuyez sur cette icône pour ouvrir un menu contextuel (à droite) et accéder :
    • aux mots clés disponibles ;
    • aux indicateurs de performance clé (KPI) ;
    • aux commandes Groupes, Configurer pour sélectionner les groupes à afficher da,s vos vues Liste. Ces commandes sont disponibles dans les vues Liste suivantes : Compte, Contact, Prospect, Opportunité et Ticket.
  • Cette icône indique que vous pouvez développer et réduire les volets afin d'afficher ou de masquer les détails.
  • Cette icône indique que vous pouvez ajouter une fiche dans CRM. Elle est disponible dans toutes les vues Liste.
  • Lorsque le bouton Actualiser est disponible, appuyez dessus pour actualiser immédiatement les informations. Si vous souhaitez que l'actualisation ne s'applique qu'à une seule fiche, sélectionnez cette dernière. Lorsqu'aucune fiche n'est sélectionnée, ce sont les données de l'ensemble du groupe qui sont actualisées.
  • Le texte souligné dans les vues Détail indique une action hypertexte (par exemple, l'accès à une URL ou à un compte).

Actions rapides

  • Dans les vues Liste, appuyez sur l'icône pour développer la liste des actions rapides possibles sur cette fiche.
  • Après un appel téléphonique ou un e-mail, une vue vous permettant de saisir des notes et de les enregistrer dans l'historique s'affiche.
  • En mode Liste, cliquez sur l'icône de la fiche pour accéder aux raccourcis des fonctions fréquentes. (Certaines icônes d'actions rapides peuvent ne pas être disponibles. Par exemple, dans Mes activités, l'action Refuser ne sera pas disponible si vous avez déjà accepté la réunion.) Les sections Mes activités et Pièces jointes sont disponibles dans Saleslogix 8.0 et versions ultérieures
  • L'action d'affichage de l'adresse vous permet d'accéder à l'application de mappage native sur les appareils pris en charge.

Rechercher

  • La recherche peut s'effectuer de différentes manières :

    • Dans le menu Global, vous pouvez effectuer une recherche sur tous les comptes associés, contacts, prospects, opportunités, activités, notes/historique et tickets. Saisissez davantage d'informations pour affiner votre recherche.
    • Dans le menu contextuel, vous pouvez filtrer les résultats de la vue Liste avec des mots clés prédéfinis
    • Pour chaque entité, filtrez les résultats de la vue Liste depuis la barre de recherche à l'aide du champ spécifié ci-après.
