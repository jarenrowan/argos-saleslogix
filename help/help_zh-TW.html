快速提示

  • 點選此圖示移至“全域”功能表，以進行下列操作：
    • 使用“快速搜尋”進行搜尋（Saleslogix 8.0 及更新版本）。
    • 使用“快速動作”快速新增客戶/聯絡人。
    • 使用“移至”存取行動用戶端的所有記錄類型。
    • 使用“組態功能表”變更“移至”功能表中的項目，並指定項目顯示的順序。“組態功能表”畫面的第一個清單項目將成為預設檢視，並在登入後首先載入。
    • 使用“設定”可清除儲存的認證、清除儲存空間及存取錯誤日誌。
    • “設定”下方的“清除儲存空間”選項只是一個疑難排解裝置，用於清除儲存在本機的設定。
    • 尋找“說明”。
    • 登出。
  • 點選此圖示移至“操作”（右側）功能表。選項會依記錄類型而有不同。
    • 可用的 # 標籤
    • 關鍵效能指標 (KPI)
    • 展開“群組”、“組態”，以選取要在清單檢視上顯示的群組。此功能可在客戶、聯絡人、銷售線索、商機和記錄單清單檢視上使用。
    • 選取群組配置。此功能可在客戶、聯絡人、銷售線索、商機和記錄單清單檢視上使用。所選取的配置會成為之後返回此記錄類型時的預設值。
      • 摘要配置會在 1 欄顯示 4 個欄位。這是預設值。
      • 詳細資料配置會根據裝置的寬度，顯示 1 至 3 欄的欄位。
        備註： 如果您有 CRM 網站或 Windows 用戶端的存取權，可以修改這些行動摘要和詳細資料配置中一些欄位的顯示方式。
        • 在網頁用戶端“客戶”、“聯絡人”、“銷售線索”、“商機”或“記錄單”清單檢視的群組標籤上按一下滑鼠右鍵，按一下“編輯”，再按一下“配置”標籤：
      • 拖放欄以變更其順序。
      • 以綠色文字顯示貨幣欄位：選取“潛力”欄，按一下“編輯”，展開“格式類型”欄位下拉式清單，然後選取“貨幣”，按一下“確定”。
      • 將電話號碼顯示為可按一下的連結：選取“電話號碼”，按一下“編輯”，展開“格式類型”欄位下拉式清單，然後選取“電話號碼”，按一下“確定”。
      查看網站或 Windows 用戶端說明，以取得編輯群組的詳細資訊。

    搜尋

    • 搜尋的方法有很多種：

      • 在“全域”功能表中找尋所有相關客戶、聯絡人、銷售線索、商機、活動、備註/歷程記錄和記錄單。輸入較多文字以提高搜尋的準確性。
      • 在“操作”功能表，使用預先定義的 # 標籤關鍵字，篩選清單檢視的結果。
      • 針對每個實體，在查閱列使用下方指定的欄位篩選清單檢視結果。

    • 客戶

      搜尋欄位

      • 客戶名稱 : 開頭為

    • 聯絡人

      搜尋欄位

      • 名字 : 開頭為
      • 姓氏 : 開頭為
      • NameLF : 包含

    • 銷售線索

      搜尋欄位

      • 公司名稱 : 開頭為
      • 名字 : 開頭為
      • 姓氏 : 開頭為
      • NameLF : 包含
