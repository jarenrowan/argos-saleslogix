Quick Tips

  • Tap this icon to go to the Global menu, where you can:
    • Search using SpeedSearch (Saleslogix 8.0 and later).
    • Use Quick Actions to quickly add an Account/Contact.
    • Use Go To to access all of the record types in the Mobile Client.
    • Use Configure Menu to change the items in the Go To menu and specify the order in which they appear. The first item in the list on the Configure Menu screen becomes your default view and loads first after you log on.
    • Use Settings to clear saved credentials, clear storage and access error logs.
      • The "Clear Storage" option under Settings is strictly a trouble-shooting device to clear locally stored configuration.
    • Find Help.
    • Log out.
  • Tap this icon to go to the Contextual (right) menu to access:
    • Available hashtags
    • Key Performance Indicators (KPIs)
    • Expand Groups, Configure to select the groups to show on your list views. This is available on Account, Contact, Lead, Opportunity, and Ticket list views.
  • This icon indicates that you can expand and collapse panels to show/hide additional details.
  • This icon indicates that you can add a new record to CRM. It is available on all list views.
  • When the Refresh button is available, tap it to immediately refresh information. Select a record to limit the refresh to that record. When no record is selected, data for the entire group is refreshed.
  • Underlined text in detail views signifies a link action (such as browsing to a URL or jumping to an Account).

Quick Actions

  • In list views, tap the avatar to expand Quick Actions for that record.
  • Following phone call or email actions, a view enabling you to type notes and save to history displays.
  • While in the list view, click the record icon to see and access short-cuts to frequently-used features. (Some quick action icons may be unavailable. For example, in My Activities, the Decline action will be unavailable if you have already accepted the meeting.) My Activities and Attachments are available in Saleslogix 8.0 and later.
  • The "View address" action takes you to the native map application on supported devices.

Search

  • There are multiple ways to search:

    • From the Global menu, find all related accounts, contacts, leads, opportunities, activities, notes\history, and tickets. Enter more text to improve the accuracy of your search.
    • From the Contextual menu, filter list view results using pre-defined hashtag keywords.
    • For each entity, filter list view results from the lookup bar using the fields specified below.
