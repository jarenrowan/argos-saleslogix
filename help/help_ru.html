Быстрые подсказки

  • Нажмите этот значок, чтобы перейти в глобальное меню, где вы сможете:
    • Выполнить поиск с помощью функции Быстрого поиска (Saleslogix 8.0 или более поздней версии).
    • Использовать Быстрые действия для удобного добавления учетной записи или контакта.
    • Воспользоваться функцией Перейти для доступа ко всем типам записей в мобильном клиенте.
    • Выполнить Настройку меню для изменения объектов в меню перехода и выбора порядка их отображения. Первый объект в списке "Настройка меню" становится представлением по умолчанию и загружается первым при входе в систему.
    • Использовать Настройки для удаления сохраненных учетных данных, очистки хранилища и журналов ошибок доступа.
      • Действие очистки хранилища в настройках используется исключительно для устранения проблем и позволяет очистить сохраненные локально настройки.
    • Ознакомиться со Справкой.
    • Выйти из системы.
  • Нажмите этот значок, чтобы перейти в контекстное меню (справа) и получить доступ к:
    • Доступным хэштегам
    • Ключевым показателям эффективности
    • Разверните Группы, выполните Настройку для выбора групп, которые отображаются в списках. Эта возможность доступна в списках учетных записей, контактов, потенциальных и возможных клиентов и билетов.
  • Этот значок показывает, что вы можете развернуть и свернуть панели, чтобы показать или скрыть дополнительные сведения.
  • Этот значок показывает, что вы можете добавить новую запись в CRM. Он доступен при просмотре всех списков.
  • Если кнопка "Обновить" активна, нажмите ее для мгновенного обновления информации. Выберите запись, чтобы обновить только ее. Если запись не выбрана, будет обновлена вся группа.
  • Подчеркнутый текст при просмотре сведений означает ссылку на действие (например, просмотр веб-страницы или переход в учетную запись).

Быстрые действия

  • При просмотре списков нажмите аватар, чтобы развернуть список быстрых действий для этой записи.
  • Переход к просмотру телефонных вызовов или действий электронной почты, где вы можете вводить заметки и просматривать историю.
  • При просмотре списков щелкните значок записи, чтобы открыть доступ к ярлыкам часто используемых функций. (Некоторые значки быстрых действий могут быть недоступны. Например, в разделе "Мои действия" действие "Отклонить" будет недоступно, если вы уже подтвердили свое участие.) "Мои действия" и "Вложения" доступны в версиях Saleslogix 8.0 и выше.
  • Действие "Просмотр адреса" переключает вас во встроенное приложение просмотра карты, если устройство поддерживает его.

Поиск

  • Существует несколько вариантов поиска:

    • В главном меню доступен поиск всех связанных учетных записей, контактов, потенциальных и возможных клиентов, действий, заметок/истории и билетов. Чтобы повысить точность поиска, вводите более длинные запросы.
    • В контекстном меню доступна фильтрация списков результатов с помощью предустановленных ключевых хэштегов.
    • Результаты поиска можно отфильтровать по следующим параметрам:

  • Учетные записи

    Поля поиска

    • Имя учетной записи: начинается с

  • Контакты

    Поля поиска

    • Имя: начинается с
    • Фамилия: начинается с
    • Имя/фамилия: содержит

  • Возможные клиенты

    Поля поиска

    • Название компании: начинается с
    • Имя: начинается с
    • Фамилия: начинается с
    • Имя/фамилия: содержит

  • Билеты

    Поля поиска

    • Номер: начинается с
    • Индекс: начинается с
    • Тема: начинается с
    • Имя учетной записи: начинается с

  • Действия

    Поля поиска

    • Тема: содержит

  • Мои действия

    Поля поиска

    • Тема: содержит

  • Мои вложения

    Поля поиска

    • Описание вложения: содержит

  • Заметки/История

    Поля поиска

    • Тема: содержит

  • Потенциальные клиенты

    Поля поиска

    • Имя потенциального клиента: начинается с
    • Имя учетной записи: начинается с

  • Продукты (от возможных клиентов)

    Поля поиска

    • Продукт: начинается с
