Быстрые подсказки

  • Коснитесь этого значка, чтобы перейти в глобальное меню, где вы сможете:
    • Использовать поиск с помощью функции SpeedSearch (Saleslogix версии 8.0 или более поздней).
    • Использовать Быстрые действия для удобного добавления учетной записи или контакта.
    • Воспользоваться функцией Перейти для доступа ко всем типам записей в мобильном клиенте.
    • Настроить меню для изменения пунктов меню перехода и их порядка. Первый объект в списке "Настройка меню" становится представлением по умолчанию и загружается первым при входе в систему.
    • Использовать Настройки для удаления сохраненных учетных данных, очистки хранилища и журналов ошибок доступа.
    • Действие очистки хранилища в настройках используется исключительно для устранения проблем и позволяет очистить сохраненные локально настройки.
    • Поиск справки.
    • Выйти из системы.
  • Коснитесь этого значка для перехода к контекстному (правому) меню Доступные опции зависят от типа записи.
    • Доступные хэш-теги
    • Ключевые показатели эффективности
    • Разверните Группы, нажмите Настроить для выбора групп, которые отображаются в списках. Эта возможность доступна в списках учетных записей, контактов, потенциальных и возможных клиентов и билетов.
    • Выберите представление групп. Эта возможность доступна в списках учетных записей, контактов, потенциальных и возможных клиентов и билетов. Выбранный режим представления будет использоваться как режим по умолчанию для записей этого типа.
      • Режим сводного представления показывает четыре поля в одном столбце. Это режим по умолчанию.
      • Режим подробного представления показывает поля в 1–3 столбцах в зависимости от ширины экрана устройства.
        Примечание: В веб- и Windows-версиях клиента CRM есть возможность изменить отображение некоторых полей в сводном и подробном мобильных представлениях.
        • В списке учетных записей, контактов, потенциальных клиентов, возможных клиентов или билетов веб-клиента щелкните правой кнопкой вкладку группы, затем щелкните Изменить и выберите вкладку Представление.
      • После этого вы сможете, используя перетаскивание, поменять порядок столбцов.
      • Чтобы выводить денежные суммы зеленым цветом, выберите столбец Потенциал, щелкните Изменить, в раскрывающемся списке Формат выберите значение Валютаи щелкните ОК.
      • Чтобы выводить телефонные номера в виде ссылок, выберите столбец Телефон, щелкните Изменить, в раскрывающемся списке Формат выберите Телефон и щелкните ОК.
      Узнать больше о редактировании групп вы можете из справки к веб- и Windows-клиентам CRM.

    • В представлении списка коснитесь значка записи для доступа к ярлыкам часто используемых функций. Доступные опции отличаются в зависимости от типа записи. "Мои действия" и "Вложения" доступны в версиях Saleslogix 8.0 и выше.

    Поиск

    • Существует несколько вариантов поиска:

      • В главном меню доступен поиск всех связанных учетных записей, контактов, потенциальных и возможных клиентов, действий, заметок/истории и билетов. Чтобы повысить точность поиска, вводите более длинные запросы.
      • В контекстном меню доступна фильтрация списков результатов с помощью предустановленных ключевых хэштегов.
      • Результаты поиска можно отфильтровать по следующим параметрам:

    • Учетные записи

      Поля поиска

      • Имя учетной записи: начинается с

    • Контакты

      Поля поиска

      • Имя: начинается с
      • Фамилия: начинается с
      • Имя/фамилия: содержит

    • Возможные клиенты

      Поля поиска

      • Название компании: начинается с
      • Имя: начинается с
      • Фамилия: начинается с
      • Имя/фамилия: содержит

    • Билеты

      Поля поиска

      • Номер: начинается с
      • Индекс: начинается с
      • Тема: начинается с
      • Имя учетной записи: начинается с

    • Действия

      Поля поиска

      • Тема: содержит

    • Мои действия

      Поля поиска

      • Тема: содержит

    • Мои вложения

      Поля поиска

      • Описание вложения: содержит

    • Заметки/История

      Поля поиска

      • Тема: содержит

    • Потенциальные клиенты

      Поля поиска

      • Имя потенциального клиента: начинается с
      • Имя учетной записи: начинается с

    • Продукты (от возможных клиентов)

      Поля поиска

      • Продукт: начинается с
